SimpleMusicVisualizer
=====================

A simple music visualizer

Running from Source
===================
Install the following dependencies:
<table>
<tr><td>Python 3.x</td></tr>
<tr><td>NumPy</td></tr>
<tr><td>matplotlib</td></tr>
<tr><td>librosa</td></tr>
<tr><td>vlc</td></tr>
</table>

The program can be run with `python main.py`
